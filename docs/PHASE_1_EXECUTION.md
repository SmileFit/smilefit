# SmileFit – Phase 1 Execution Plan (€1,000)

This document defines the ONLY work to be executed for Phase 1.

Time budget: ~20–22 development hours  
Goal: Professional marketing website  
Not a platform. Not an app.

---

## Pages (5 total)

1. Home
2. How It Works
3. Classes (static showcase)
4. Instructors (static showcase)
5. Contact (email or WhatsApp)

---

## Included Features

- Responsive layout (desktop + mobile)
- Header + footer navigation
- Static content stored in TypeScript files
- Contact form that sends email (no storage)
- Basic SEO metadata
- Deployment on Vercel

---

## Explicit Exclusions

- Authentication
- Booking or reservations
- Payments
- Dashboards
- CMS
- Database
- Analytics
- Advanced SEO
- CI/CD pipelines
- Accessibility audits beyond basic semantics

Any feature not listed above is Phase 2 by default.

---

## Quality Bar

- Clean, professional appearance
- Works on modern mobile + desktop browsers
- No critical bugs
- Fast load times

---

## Time Allocation (Hard Cap)

- Setup & layout: 3 hrs
- Home page: 5 hrs
- Remaining pages: 7 hrs
- Contact form: 2 hrs
- Polish & deploy: 3 hrs

Exceeding this budget requires explicit approval.
